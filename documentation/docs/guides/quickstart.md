# Quick Start Guide

Get your development environment up and running in minutes.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Docker** (version 20.10+)
- **Docker Compose** (version 2.0+)
- **Python** (version 3.12+)
- **Git**
- **NodeJS** (version 24+)

### Verify Installation

```bash
docker --version
docker-compose --version
python --version
git --version
```
